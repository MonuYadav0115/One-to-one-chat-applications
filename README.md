# Real-Time Chat Application

## Features

- Authentication (JWT)
- One-to-One Chat
- Socket.IO Real-Time Messaging
- Online/Offline Status
- Typing Indicator
- Delivered/Seen Messages
- Message Search
- Delete Messages
- Unread Message Count
- Logout

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Socket.IO Client

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT Authentication

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm run dev
